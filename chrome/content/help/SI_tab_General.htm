Choose 'Save Images' language Select a language that will be used by 'Save Images'. Firefox must be shutdown and restarted for this option to be initialized.
Open Save Images Options dialog box before saving If checked, the Save Images Options dialog box will be displayed before the save operation takes place. This gives you the opportunity to change options before each save. Clicking the Cancel button will stop the saving operation completely. (Only available in Windows)
Show save confirmation popup After the save operation, a notification will be displayed at the bottom right of the screen indication how many images were saved and if there were any errors.
Only if an error occurs The notification will only be displayed if an error occurs while saving images.
Display notification popup at bottom/right corner of Firefox window The notification will be displayed at the bottom/right corner. If unchecked, the notification will be displayed beneath the "Save Images" toolbar button if it has been added to the toolbar.
Notification popup display time (seconds) The notification will be displayed for the number of seconds specified.
Show a detailed list of saved/not saved images Opens a window showing information about the last save operation.
Only when images are saved The window of save details is only shown if images are actually saved.
Save background images Background images will be saved. Any duplicate images will be not be saved.
Save image under the mouse cursor Using the keyboard combination specified under Keyboard Shortcuts, the image under the mouse cursor will be saved, using the options set for Filters and Save.
Ignore "Filters" set in "Options" The image will be saved regardless of the options set in Filters.
Number of folders/file names to remember Set the number of folders/file names to be saved from the folder/file name drop down menus.
   
Opening Multiple Links/Images Set the delay and number of links/images that will be opened with each interval.
Frame height when multiple links are opened in one tab Set the height each frame will be when multiple links are opened in a single tab.
Time delay between opening links/images (seconds) Set the time in seconds between when groups of links/images will be opened. If set to zero, all links/images selected will be opened at once.
Number of links/images to open at once Set the number of links/images that will be opened at each time interval. If set to zero, all links/images selected will be opened at once.
Automatically switch to new tab (when opening multiple links) If multpiple links or image links are opened in 1 tab and this option is checked, then the newly created tab will be selected, displaying the created web page.
Bypass opening 'Select Links' dialog box With this option checked, the "Select Links" dialog box will not be displayed before opening the links/images
