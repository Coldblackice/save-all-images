Use to specify a folder and file name by which the images will be saved. A number will either be prefixed or suffixed to the file name, beginning with Start Value and incremented each time by the value given in Increment By. The images file name may also be appended to the file name given.

Filename Options  
Save Folder Specify a folder where to save the images. Click the button next to the drop down list to browse for a folder. A previously used folder may be selected from the drop down list.
(Browse for a folder) Click to display a window where you may select a folder to use.
(Clear folder list) Click to clear all the folders from the Use Folder list. (Not undo-able)
Filename Specify a file name to use. A previously used file name may be selected from the drop down list.
(Clear file name list) Click to clear all the file names from the Use filename list. (Not undo-able)
Retain original filename If checked, the images file name is appended to whatever file name you enter.
Prepend original file name with The text entered will be added in front of the original file name if the Retain original filename option is checked.
   
Prefix/Suffix Number A number is added to the beginning or end of the file name you enter.
Start Value A number from which to start numbering the saved images.
Length The number of digits to use for the numbering. eg. a length of 3 would be 001 etc.
Increment By A number by which to increment the Start Value and each successive number.
Prefix / Suffix Specify whether the numbers should be added to the beginning or end of the file name.
   
Default Click the "Default button" to save the "Prefix/Suffix Number" options for future use.
