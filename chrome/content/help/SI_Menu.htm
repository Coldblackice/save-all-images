There are 3 menus from which the various "Save Images" functions can be accessed - the Tools menu, the context menu and the toolbar button dropdown menu (if added to a toolbar). Any of the menu items in any of the 3 menus may be shown or hidden by checking or unchecking the appropriate checkbox next to the menu item in the Options dialog box.

At the bottom are 2 options which can be set to hide either the "Save Images" Tools menu or the "Save Images" context menu.

Pale Moon does not need to be restarted for the changes to be operational.

To add the Save Images toolbar button to a toolbar, right click on a toolbar and select Customize... or select from the Pale Moon menu View -> Toolbars -> Customize.... Find the Save Images button and drag-and-drop it onto a toolbar to add the button to the toolbar.
