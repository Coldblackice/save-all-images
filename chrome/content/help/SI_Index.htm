"Save Images" saves the images from the current tab web page, tabs to the left or right or from all tabs, from the Firefox cache, to a specified location, with either the images original file name or a file name that you specify. The size and types of images saved may be chosen, with options if the file size or image dimensions are unknown or if duplicate files are found. A regular expression may also be applied to the images URL for further filtering.

There are also menu items to either open all or selected links in a single tab or multiple tabs. The menu item "Save Details" displays information pertaining to the last save operation.

"Save Images" can be accessed from the Firefox Tools menu, the context menu (right click on web page) and from a button placed on a toolbar.

To place the toolbar button on a toolbar, right click a toolbar and select "Customize...". A "Customize Toolbar" dialog box is displayed showing the buttons that can be placed on a toolbar. Select the "Save Images" button and drag it onto a toolbar. Close the dialog box.

If the toolbar button is present, clicking it will save the images according to what option is selected for Default Save Action in the Options Dialog Box Save tab. If the Ctrl key is held down while clicking the toolbar button, the images will be saved with the images filename (as if setting Use images filename was selected), irrespective of what setting in File Name is selected. If the Ctrl and Alt keys are held down while clicking the toolbar button, a Filename Options Dialog Box will be displayed, prompting you for a file name (as if setting Prompt for filename/Once per page was selected), irrespective of what setting in File Name is selected.
