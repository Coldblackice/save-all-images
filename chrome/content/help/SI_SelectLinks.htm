Use the Select Links dialog box to make a selection of links that you want opened in either a single tab or multiple tabs.

- Select All Click to check all the items in the list.
- Select None Click to uncheck all the items in the list.
- Check Selected Click to check all the items in the list that are currently selected.
- Invert Checked Items Click to invert the checked items in the list.
- Find Phrase Click to display a dialog box to enter a phrase. All items in the list containing the phrase will be selected.
- Move Item Up Click to move the currently selected item up 1 position in the list.
- Move Item Down Click to move the currently selected item down 1 position in the list.
