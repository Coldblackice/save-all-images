Enable/disable file size and image dimension restrictions, the action to take when duplicate file names are found, the image types to save and whether to use a regular expression to filter the images that are found on the web page.

Enable Size Limits Select to be able to use the various size limit options. The size limit options only work if the file is in Pale Moon's cache.
File Size Select to use and to be able to set the minimum and maximum file sizes in bytes (1024 bytes = 1kB).
Min Set the minimum size a file must be to be saved.
Max Set the maximum size a file can be to be saved. If zero, all files larger than the minimum size will be saved.
If unknown Select one of the settings below if the image's file size is unknown.
Skip The image will not be saved.
Prompt A message box is displayed prompting you whether to save the specified image or not.
Save The image will be saved.
   
Dimensions Select to use and to be able to set the minimum width and height.
Min Width Set the minimum width an image must be to be saved.
Max Width Set the maximum width an image must be to be saved.
Min Height Set the minimum height an image must be to be saved.
Max Height Set the maximum height an image must be to be saved.
If unknown Select one of the settings below if the image's dimensions are unknown.
Skip The image will not be saved.
Prompt A message box is displayed prompting you whether to save the specified image or not.
Save The image will be saved.
   
Image Types to save Select one of the settings below to choose what type of images will be saved.
All images All image types will be saved.
Selection A combination of any of the following image types will be saved.
JPG / JPEG images Images with either a JPG or JPEG extension will be saved.
PNG images Images with a PNG extension will be saved.
GIF images Images with a GIF extension will be saved.
BMP images Images with a BMP extension will be saved.
   
Save image as JPG if no extension Select this option to save images that do not have an extension. The image will be given a JPG extension when saved.
   
Regular Expressions Select this option and provide a Regular Expression that will be used to filter the images that will be saved. This can be used in conjunction with Regular Expression file name under File Names in the Save tab. Read this Regular Expressions document for more details.
   
Only save images from the same domain/server as the selected tab's URL If the image has a domain that is different to the selected tab's URL, the image will not be saved.
