The following menu items can be accessed from either the Firefox Tools menu, the context menu (right click on web page) or from the Save Images button which can be placed on a toolbar. What menu items are displayed can be selected in the Options Dialog Box under the Menu Items tab.

Save images from current tab The images from the current tab are saved.
Save images from tabs to the LEFT The images from all tabs to the left of the current tab are saved. (This may include the current tab if the option is set)
Save images from tabs to the RIGHT The images from all tabs to the right of the current tab are saved. (This may include the current tab if the option is set)
Save images from ALL tabs The images from all the tabs are saved.
   
Image Links  
Open ALL image links in 1 tab Displays the Select Links Dialog Box and lists all image links, and if Ok is clicked, the checked links are opened in 1 tab.
Open SELECTED image links in 1 tab Displays the Select Links dialog box and lists the selected image links, and if Ok is clicked, the checked links are opened in 1 tab.
Open ALL image links in TABS Displays the Select Links dialog box and lists all image links, and if Ok is clicked, the checked links are opened in multiple tabs.
Open SELECTED image links in TABS Displays the Select Links dialog box and lists the selected image links, and if Ok is clicked, the checked links are opened in multiple tabs.
   
Links  
Open ALL links in 1 tab Displays the Select Links dialog box and lists all links, and if Ok is clicked, the checked links are opened in 1 tab.
Open SELECTED links in 1 tab Displays the Select Links dialog box and lists the selected links, and if Ok is clicked, the checked links are opened in 1 tab.
Open ALL links in TABS Displays the Select Links dialog box and lists all links, and if Ok is clicked, the checked links are opened in multiple tabs.
Open SELECTED links in TABS Displays the Select Links dialog box and lists the selected links, and if Ok is clicked, the checked links are opened in multiple tabs.
   
Options Displays the Options Dialog Box.
Save Details Opens a window showing information about the last save operation.
Help Displays this page in a new tab.
