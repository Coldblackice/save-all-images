Various options may be set to change how the images are saved, what is saved and where they are saved.

Keyboard shortcuts may be assigned to the various menu items, as well as choosing what menu items are displayed in the various menus.

Tabs
Filters Enable/set file size and image dimension restrictions and the image types to save.
Save Enable/set what tabs to save images from, the folder to save to and whether to use the images file name, a user entered file name or to prompt for the file name.
General A few universal options.
Keyboard Shortcuts Set keyboard shortcuts for the various menu items (requires a restart of Pale Moon to become operational).
Menu Items Set the menu items that will be displayed in the various menus. No restart is required to activate the changes.

The Restore Defaults button at the bottom resets all options to their settings when the add-on was installed. There are also 2 buttons to Export and Import the options to and from a file.
