Enable / set what tabs to save images from, the folder to save to and whether to use the images file name, a user entered file name or to prompt for the file name.

Default Save Action Select one of the settings to be what happens when the toolbar button is clicked.
Current tab Only images from the currently displayed tab are saved.
Left of current tab The images from all tabs to the left of the current tab are saved. (This may include the current tab if the option is set)
Right of current tab The images from all tabs to the right of the current tab are saved. (This may include the current tab if the option is set)
All tabs Images from all the tabs are saved.
Include the Current tab when saving images (Left/Right) If checked, images from the current tab will also be saved if the Default Save Action is set to Left of current tab or Right of current tab.
   
Folder Options Set various options that will apply to the folder that is to be saved to.
Folder Select a folder where the images will be saved.
Use Firefox download folder Select to use the folder set in Firefox for downloads. To set this folder, go to Tools->Options..., click the Main icon and then browse for a folder for the Save Files to option. If this folder is either "Desktop" or "My Documents", the Use Firefox downloads folder will be unavailable.
Use last download folder Select to save to the last folder that was used to save something from Firefox.
Use Folder Specify a folder where to save the images if the option above is not checked. Click the button next to the text box to browse for a folder. A previously used folder may be selected from the drop down list.
(Browse for a folder) Click to display a window where you may select a folder to use.
(Clear folder list) Click to clear all the folders from the Use Folder list. (Not undo-able)
Append the tab title to 'Save Folder' The title of the currently selected Firefox tab will be appended to the 'Save Folder'.
Append image URL folder structure to 'Save Folder' A folder structure will be created based on the URL of the image being saved and appended to the 'Save Folder'.
Tree structure Folders will be created using a cascading tree structure.
Single folder A single folder will be created from the URL by joining the separate folders together with underscores.
Append the date to the folder The date is appended as part of the folder name.
Append the time to the folder The time is appended as part of the folder name.
Include seconds The seconds will be appended as part of the time that is added to the folder name.
   
File Name Specify what file names to use when images are saved.
Use image's filename The image's file name in the web page is used for saving.
Use filename Enter a file name to be used when the image is saved. A previously used file name may be selected from the drop down list.
(Clear file name list) Click to clear all the file names from the Use filename list. (Not undo-able)
Prompt for filename A dialog box is displayed prompting you for a file name. See Filename Options Dialog Box for more details.
Once per page The dialog box will only be displayed at the start of the save operation.
For each image The dialog box will be displayed for every image that is to be saved.
Regular Expression file name This option can be use in conjunction with Regular Expressions in the Filters tab to provide a file name that will be used when the images are saved. Read this Regular Expressions document for more details.
   
Duplicate Files Select one of the settings below to decide how duplicate file names are handled.
Ignore All files with the same file names will not be saved.
Prompt A "Save As" dialog box will be displayed asking for you to specify a file name.
Overwrite A file with the same file name will be overwritten by the new file.
Number Files with the same file name will have a number appended to the file name.
Length Set the number of digits to have in the number.
   
Close tab after saving If images are saved from the tab and this option is checked, the tab will be closed after the save operation is complete.
Prompt about closing tab A message will be displayed asking if the tab must be closed.
   
Clear "Use Folder" list when Firefox exists The Use Folder list will be cleared when Firefox is closed. N.B. Only works if Firefox is closed by accessing the File menu and selecting Exit / Quit.
Clear "Use filename" list when Firefox exists The Use filename list will be cleared when Firefox is closed. N.B. Only works if Firefox is closed by accessing the File menu and selecting Exit / Quit.
